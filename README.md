# Gold Nanoparticles for Drug Delivery in Lung Cancer

## Overview
This repository contains research materials, analysis, and supporting resources for the study:

**“Gold Nanoparticles for Drug Delivery in Lung Cancer: A Comprehensive Analysis of Active and Passive Targeting Mechanisms via the Enhanced Permeability and Retention Effect.”**

The work explores how gold nanoparticles (AuNPs) can be used as nanocarriers for targeted drug delivery in lung cancer therapy.

## Research Focus

The project investigates two key mechanisms:

1. Passive Targeting  
   - Enhanced Permeability and Retention (EPR) effect  
   - Tumor vascular permeability

2. Active Targeting  
   - Ligand–receptor binding  
   - Functionalized nanoparticle surfaces

## Topics Covered

- Gold nanoparticle synthesis
- Nanoparticle characterization techniques
- Drug loading strategies
- Controlled drug release mechanisms
- Tumor microenvironment interactions
- Mathematical modeling of nanoparticle transport
