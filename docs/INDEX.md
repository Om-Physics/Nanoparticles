## Getting Started

Install dependencies, then run the project in five steps.

```bash
pip install -r requirements.txt       # Step 1: Install dependencies
python src/generate_figures.py        # Step 2: Generate all six figures
python src/data_export.py             # Step 3: Export reference datasets
# Step 4: Open simulation/Gold_NP_Simulation.html in any browser
python examples/quick_start.py        # Step 5: Run the worked example
pytest tests/ -v                      # Step 6: Run unit tests
```

## Citation

When using this framework in publications, include the following statement in the methods section.

"Computational biodistribution, pharmacokinetic, toxicity, and efficacy analyses were performed using a custom Python framework implementing two-compartment pharmacokinetic models, EPR accumulation differential equations, Korsmeyer-Peppas drug release kinetics, and CEM43 photothermal dose calculations. Parameters were derived from published literature on PEGylated and antibody-conjugated gold nanoparticle systems. All results represent computational projections and not experimental measurements. Code and parameter documentation are available at github.com/Om-Physics/Nanoparticles."
